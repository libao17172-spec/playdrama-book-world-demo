# 3D世界视觉验收基准

`naval-world-target-v1.png` 由 GPT Image 2 根据项目参考图生成，作为《纳瓦尔宝典》3D互动世界下一版的画面验收基准。

后续场景制作必须满足以下要求：第三人称人物在画面中占约三分之一高度；建筑具有真实比例、连续街区层次和可辨认材质；道路、树木、店铺橱窗、灯光与远景共同建立空间深度；知识道具自然融入环境；不得再使用胶囊人物、默认几何体、灰盒建筑或悬浮水晶承担主要画面。

推荐制作链路为：GPT Image 2生成整体验收图和关键资产多视角图，TRELLIS.2或Hunyuan3D生成关键GLB资产，Blender完成自动减面、材质整理与碰撞代理，Three.js负责运行时加载、人物控制、交互和知识节点。
